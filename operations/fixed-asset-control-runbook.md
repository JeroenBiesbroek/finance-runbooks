# Fixed Asset Control Runbook

**Owner:** Finance Lead

## When to use
When assets need to be added, deprecated, disposed of, or reviewed in Exact Online.

## Asset addition
1. Confirm asset meets capitalization threshold (> EUR 500 excl. VAT)
2. Gather: description, acquisition date/cost, useful life, depreciation method, GL accounts
3. Finance Lead approval
4. Enter in Exact Online → Assets → Asset register
5. Attach supporting documentation

## Monthly depreciation
1. Exact Online → Assets → Depreciation
2. Select current period, run calculation (preview first)
3. Review output:
   - All active assets included
   - Amounts reasonable
   - Consistent with prior period
4. Post depreciation
5. Document in close checklist

## Asset disposal
1. Determine disposal details (date, proceeds, gain/loss)
2. Finance Lead approval
3. Process in Exact: remove from active register, post disposal journal
4. Verify GL entries
5. Attach supporting documentation

## Quarterly asset review
1. Generate asset register from Exact
2. For each asset: does it still exist? Useful life appropriate? Impairment indicators?
3. Document findings
4. Process changes (Finance Lead approval)

## Controls
- CTRL-FA-001: Register completeness (monthly)
- CTRL-FA-002: Depreciation review (monthly)
- CTRL-FA-003: Addition/disposal approval (per transaction)
