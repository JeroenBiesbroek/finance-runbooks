# Journal Handling Runbook

**Owner:** Finance Lead

## When to use
When manual, recurring, accrual, correcting, or reclassification journal entries need to be created and posted in Exact Online.

## Journal types

| Type | Approval |
|---|---|
| Recurring | Finance Lead (template approval) |
| Accrual / Deferral | Finance Lead |
| Correcting | Finance Lead |
| Reclassification | Finance Lead |
| Ad-hoc | Finance Lead |

## Procedure

### 1. Prepare
1. Determine journal type
2. Prepare supporting documentation
3. Verify entry balances (debits = credits)
4. Verify GL accounts exist, period is open

### 2. Review
1. Self-review for accuracy
2. Cross-reference with supporting docs
3. Verify VAT treatment (if applicable)

### 3. Approve
1. Submit to Finance Lead
2. Attach all supporting documentation
3. Wait for explicit approval

### 4. Post in Exact Online
1. Financial → Journal entries
2. Enter date, accounts, description, amounts, VAT codes
3. Attach supporting documentation
4. Post

### 5. Verify
1. Entry appears in journal listing
2. Account balances updated correctly
3. Correct period

## Recurring entry process
1. Review template at start of each period
2. Verify amounts still correct
3. Process all recurring entries
4. Review posted entries against template

## Common issues

| Issue | Resolution |
|---|---|
| Period closed | Request reopen (Finance Lead + CFO) or post to next period |
| Account missing | Create account (Finance Lead approval) |
| Entry unbalanced | Correct before posting |
| Wrong period | Reverse and re-post |
